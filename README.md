# osa-observatory
 OSA-Observatory - Plateforme open source calculant l'Indice de Souveraineté Africaine (ISA) — 54 pays africains, 8 piliers stratégiques, 120 indicateurs, données historiques 2010→N-2. Collecte automatisée (WB, FMI, OMS, UIT, FAO), pipeline L1→L7, prédictions ML. Stack : PostgreSQL 15 · Python 3.11 · Streamlit · Docker.
